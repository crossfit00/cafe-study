## `회원탈퇴 Docs`

| 메소드    | 경로      | 설명   |
|--------|---------|------|
| DELETE | /member | 회원탈퇴 |

### ㅤ
## Request Header

```json
{
  "Content-Type": "application/json",
  "Authorization": "Bearer {accessToken}"
}
```

#### ㅤ
## Response

### < Success >
```json
{
  "status": "success"
}
```

* accessToken : 엑세스 토큰

#### ㅤ
### < Fail >

```json
{
  "status": "fail",
  "error": {
    "detailMessage": "인증 토큰이 없거나, 유효하지 않은 경우 발생합니다.",
    "code": "401CO000"
  }
}
```

- AccessToken이 없거나 잘못된 경우 발생

```json
{
  "status": "fail",
  "error": {
    "detailMessage": "사용 권한이 없는 경우 발생",
    "code": "403CO000"
  }
}
```

- 탈퇴 권한이 없는 경우 발생
