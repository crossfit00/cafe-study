## `주문 생성 Docs`

| 메소드  | 경로     | 설명   |
|------|--------|------|
| POST | /order | 주문생성 |

### ㅤ
## Request Header

```json
{
  "Content-Type": "application/json",
  "Authorization": "Bearer {accessToken}"
}
```

### ㅤ
## Request Body

```json
{
  "items": [
    {
      "itemId": 2,
      "quantity": 2
    },
    {
      "itemId": 1,
      "quantity": 2
    }
  ]
}
```
* itemId : 상품 아이디
* quantity: 상품 개수

#### ㅤ
## Response

### < Success >
```json
{
  "status": "success"
}
```

* accessToken : 엑세스 토큰

#### ㅤ
### < Fail >

```json
{
  "status": "fail",
  "error": {
    "detailMessage": "요청한 리소스가 존재하지 않는 경우 발생",
    "code": "404CO000"
  }
}
```

- 요청 상품이 존재하지 않을 때

<br>

```json
{
  "status": "fail",
  "error": {
    "detailMessage": "요청 주문의 상품 수보다 존재하는 상품 재고가 부족합니다.",
    "code": "400OR000"
  }
}
```

- 요청 상품이 재고가 부족할 때

<br>
